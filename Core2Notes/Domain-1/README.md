# Domain 1: Operating Systems

This folder contains my study notes for **CompTIA A+ Core 2 Domain 1 - Operating Systems**.

The notes are organized **objective by objective** according to the CompTIA A+ exam objectives. Each objective has its own Markdown file to keep the material structured, easy to review, and simple to update as I continue learning.

---

## Purpose

The goal of these notes is to:

* Understand operating system concepts required for CompTIA A+ Core 2
* Build practical Windows, Linux, macOS, and mobile OS knowledge
* Create a structured reference for future IT support work
* Document hands-on labs, commands, and troubleshooting procedures
* Review exam objectives efficiently

## How Notes Will Be Organized

Each file will contain notes related only to its corresponding objective.

Example:

* Objective1.1.md → Notes for Objective 1.1
* Objective1.2.md → Notes for Objective 1.2
* Objective1.3.md → Notes for Objective 1.3
* And so on...

This approach keeps the content modular and makes it easier to:

* Find specific topics quickly
* Review weak areas
* Update notes when needed
* Track study progress
