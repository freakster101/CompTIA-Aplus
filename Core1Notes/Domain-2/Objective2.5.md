Objective 2.5 - Common Networking Hardware Devices

Router

Definition: A device that connects different networks and directs traffic between them.
Example: A home router connecting a local network to the internet.

Switch

Definition: A device that connects devices within a network and forwards data using MAC addresses.
Example: Connecting office computers to the same network switch.

Managed Switch

Definition: A switch that allows configuration, monitoring, and advanced network management.
Example: Configuring VLANs on a managed switch.

Unmanaged Switch

Definition: A switch that operates automatically without user configuration.
Example: Plugging devices into a small office switch and using it immediately.

Access Point (AP)

Definition: A device that provides wireless network connectivity to clients.
Example: A Wi-Fi access point serving wireless users in an office.

Patch Panel

Definition: A panel used to organize and manage network cable connections.
Example: Connecting office Ethernet cables to a patch panel in a network closet.

Firewall

Definition: A device or software that filters network traffic based on security rules.
Example: Blocking unauthorized internet traffic from reaching internal systems.

Power over Ethernet (PoE)

Power over Ethernet (PoE)

Definition: A technology that delivers power and data through a single Ethernet cable.
Example: Powering an IP camera using a network cable.

PoE Injector

Definition: A device that adds electrical power to an Ethernet connection.
Example: Using a PoE injector to power a wireless access point.

PoE Switch

Definition: A network switch that provides PoE directly through its ports.
Example: A PoE switch powering multiple VoIP phones.

PoE Standards

Definition: IEEE standards that define PoE power delivery specifications.
Example: Using IEEE 802.3at (PoE+) to power a wireless access point.

---

Cable Modem

Definition: A device that provides internet access through a cable television network.
Example: Connecting to an ISP using a coaxial cable modem.

Digital Subscriber Line (DSL)

Definition: A technology that provides internet access through telephone lines.
Example: A home internet connection using DSL service.

Optical Network Terminal (ONT)

Definition: A device that converts fiber-optic signals into network signals for customer use.
Example: An ONT connecting a home to a fiber internet service.

Network Interface Card (NIC)

Definition: A hardware component that enables a device to connect to a network.
Example: An Ethernet NIC installed in a desktop computer.

Media Access Control (MAC) Address

Definition: A unique hardware address assigned to a network interface.
Example: A switch identifying a laptop by its MAC address.
