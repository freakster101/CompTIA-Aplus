Objective 2.6 - Configure Basic Wired/Wireless SOHO Networks

Internet Protocol (IP) Addressing

IPv4

Definition: A 32-bit addressing system used to identify devices on a network.
Example: 192.168.1.10

Private Address

Definition: An IPv4 address used within local networks and not routable on the internet.
Example: 192.168.1.100

Public Address

Definition: An IPv4 address that is routable on the internet.
Example: 8.8.8.8

IPv6

Definition: A 128-bit addressing system designed to replace IPv4.
Example: 2001:db8::1

Automatic Private IP Addressing (APIPA)

Definition: An address automatically assigned when a device cannot contact a DHCP server.
Example: 169.254.1.50

Static IP Address

Definition: An IP address manually assigned to a device.
Example: Assigning 192.168.1.20 to a printer.

Dynamic IP Address

Definition: An IP address automatically assigned by a DHCP server.
Example: A laptop receiving 192.168.1.105 from DHCP.

Subnet Mask

Definition: A value that identifies the network and host portions of an IP address.
Example: 255.255.255.0

Gateway

Definition: The device that forwards traffic from a local network to other networks.
Example: 192.168.1.1 serving as the default gateway for a home network.
