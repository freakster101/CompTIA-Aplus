Objective 2.1 - TCP and UDP Ports, Protocols, and Their Purposes

FTP (Port 20/21)

Definition: A protocol used to transfer files between computers over a network.
Example: Uploading website files to a web server using FTP.

SSH (Port 22)

Definition: A secure protocol used for encrypted remote access to devices.
Example: Remotely managing a Linux server using SSH.

Telnet (Port 23)

Definition: An unencrypted protocol used for remote command-line access.
Example: Connecting to a legacy network device using Telnet.

SMTP (Port 25)

Definition: A protocol used to send email messages between mail servers.
Example: Sending an email through a mail server.

DNS (Port 53)

Definition: A protocol that translates domain names into IP addresses.
Example: Resolving google.com into its IP address.

DHCP (Ports 67/68)

Definition: A protocol that automatically assigns IP configuration to network devices.
Example: A laptop receiving an IP address when joining a network.

HTTP (Port 80)

Definition: A protocol used to transfer web pages over the internet.
Example: Accessing a website that does not use encryption.

POP3 (Port 110)

Definition: A protocol that downloads email messages from a mail server to a client.
Example: Downloading emails from a mail server to Outlook.

IMAP (Port 143)

Definition: A protocol that synchronizes email between a mail server and client devices.
Example: Reading the same email on both a phone and laptop.

NetBIOS / NetBT (Ports 137-139)

Definition: Protocols used for communication and resource sharing on Windows networks.
Example: Discovering shared folders on a local Windows network.

LDAP (Port 389)

Definition: A protocol used to access and manage directory services.
Example: Authenticating users through Active Directory.

HTTPS (Port 443)

Definition: An encrypted protocol used to securely transfer web data.
Example: Logging into an online banking website.

SMB / CIFS (Port 445)

Definition: A protocol used for file and printer sharing on networks.
Example: Accessing a shared network drive in an office.

RDP (Port 3389)

Definition: A protocol used to remotely access and control Windows computers.
Example: Connecting to a work computer from home using Remote Desktop.

TCP vs. UDP

TCP (Transmission Control Protocol)

Definition: A connection-oriented protocol that provides reliable and error-checked data delivery.
Example: Downloading a file where every packet must arrive correctly.

UDP (User Datagram Protocol)

Definition: A connectionless protocol that provides faster communication without guaranteed delivery.
Example: Streaming a live video where speed is more important than perfect accuracy.
