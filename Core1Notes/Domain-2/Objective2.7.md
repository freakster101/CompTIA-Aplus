Objective 2.7 - Internet Connection Types, Network Types, and Their Characteristics

Internet Connection Types

Satellite

Definition: An internet connection that uses satellites to transmit and receive data.
Example: Providing internet access to a remote rural location.

Fiber

Definition: An internet connection that uses fiber-optic cables to transmit data using light.
Example: A home receiving gigabit internet through a fiber connection.

Cable

Definition: An internet connection that uses coaxial cable television infrastructure.
Example: A household using a cable modem for internet access.

DSL

Definition: An internet connection that uses traditional telephone lines for data transmission.
Example: A home internet service provided through a phone line.

Cellular

Definition: An internet connection that uses mobile carrier networks.
Example: Accessing the internet through a smartphone's 5G connection.

Wireless Internet Service Provider (WISP)

Definition: An internet service that delivers connectivity using wireless radio signals.
Example: A rural home receiving internet from a nearby wireless tower.

Network Types

Local Area Network (LAN)

Definition: A network that connects devices within a limited geographic area.
Example: Computers connected within a single office building.

Wide Area Network (WAN)

Definition: A network that connects devices across large geographic areas.
Example: The Internet.

Personal Area Network (PAN)

Definition: A small network used for communication between personal devices.
Example: A smartphone connected to wireless earbuds via Bluetooth.

Metropolitan Area Network (MAN)

Definition: A network that spans a city or metropolitan area.
Example: A city government's network connecting multiple offices.

Storage Area Network (SAN)

Definition: A dedicated network that provides access to centralized storage devices.
Example: Multiple servers accessing a shared storage array.

Wireless Local Area Network (WLAN)

Definition: A LAN that uses wireless technology instead of cables.
Example: Devices connected to a home Wi-Fi network.
