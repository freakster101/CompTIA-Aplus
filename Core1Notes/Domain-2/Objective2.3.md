Objective 2.3 - Services Provided by Networked Hosts

Server Roles

DNS Server

Definition: A server that translates domain names into IP addresses.
Example: Resolving www.google.com to its IP address.

DHCP Server

Definition: A server that automatically assigns IP addresses and network settings to devices.
Example: Assigning an IP address to a laptop when it joins the network.

Fileshare Server

Definition: A server that stores and shares files across a network.
Example: Employees accessing documents from a shared network drive.

Print Server

Definition: A server that manages network printers and print jobs.
Example: Multiple users printing to the same office printer.

Mail Server

Definition: A server that sends, receives, and stores email messages.
Example: Microsoft Exchange handling company email.

Syslog Server

Definition: A server that collects and stores log messages from network devices.
Example: Storing firewall and router logs for troubleshooting.

Web Server

Definition: A server that hosts websites and web applications.
Example: Hosting a company website on Apache.

Authentication, Authorization, and Accounting (AAA)

Definition: Services that verify identity, control access, and track user activity.
Example: Active Directory authenticating users before network access.

Database Server

Definition: A server that stores and manages structured data.
Example: A SQL server storing customer records.

Network Time Protocol (NTP) Server

Definition: A server that synchronizes time across network devices.
Example: Keeping all company computers on the same time.

---

Internet Appliances

Spam Gateway

Definition: A device or service that filters unwanted email messages.
Example: Blocking phishing emails before they reach users.

Unified Threat Management (UTM)

Definition: A security appliance that combines multiple security features into one device.
Example: A UTM providing firewall, antivirus, and intrusion prevention services.

Load Balancer

Definition: A device that distributes network traffic across multiple servers.
Example: Directing website visitors between several web servers.

Proxy Server

Definition: A server that acts as an intermediary between clients and other servers.
Example: Filtering employee web traffic before it reaches the internet.

---

Legacy/Embedded Systems

Supervisory Control and Data Acquisition (SCADA)

Definition: A system used to monitor and control industrial equipment and processes.
Example: Monitoring water treatment equipment from a central control station.

---

Internet of Things (IoT) Devices

Internet of Things (IoT)

Definition: Network-connected devices that collect and exchange data.
Example: A smart thermostat connected to a home Wi-Fi network.
