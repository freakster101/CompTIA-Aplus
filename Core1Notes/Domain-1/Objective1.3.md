Objective 1.3 - Mobile Device Network Connectivity and Application Support

Wireless/Cellular Data Network

Enable/Disable Wireless or Cellular Data

Definition: Turning wireless or cellular network connectivity on or off.
Example: Disabling mobile data while connected to Wi-Fi.

3G

Definition: Third-generation cellular technology used for mobile data and voice services.
Example: An older smartphone connecting to a 3G network.

4G

Definition: Fourth-generation cellular technology that provides faster mobile data speeds than 3G.
Example: Streaming videos using a 4G LTE connection.

5G

Definition: Fifth-generation cellular technology that provides high-speed and low-latency connectivity.
Example: Downloading large files quickly using a 5G network.

Hotspot

Definition: A feature that shares a mobile device's internet connection with other devices.
Example: Using a phone hotspot to provide internet access to a laptop.

Wi-Fi

Definition: A wireless networking technology that connects devices to local networks and the internet.
Example: Connecting a tablet to a home Wi-Fi network.

SIM

Definition: A physical card that stores subscriber information for cellular network access.
Example: Inserting a SIM card into a smartphone.

eSIM

Definition: A digital SIM embedded within a device that eliminates the need for a physical SIM card.
Example: Activating a cellular plan through an eSIM profile.

Bluetooth

Enable Bluetooth

Definition: Turning on Bluetooth functionality on a device.
Example: Enabling Bluetooth to connect wireless earbuds.

Enable Pairing

Definition: Making a device discoverable for Bluetooth connections.
Example: Putting a speaker into pairing mode.

Find a Device for Pairing

Definition: Searching for nearby Bluetooth-enabled devices.
Example: Locating a smartwatch from a smartphone.

Enter the Appropriate PIN Code

Definition: Providing a verification code to complete Bluetooth pairing.
Example: Entering "1234" when connecting a Bluetooth keyboard.

Test Connectivity

Definition: Verifying that paired Bluetooth devices communicate properly.
Example: Playing music through a paired Bluetooth speaker.

---

Location Services

Location Services

Definition: Features that allow applications to determine a device's location.
Example: Using a navigation app to find directions.

Global Positioning System (GPS) Services

Definition: Location services that use satellites to determine a device's position.
Example: Google Maps using GPS for navigation.

Cellular Location Services

Definition: Location services that determine position using nearby cellular towers.
Example: A smartphone estimating location when GPS is unavailable.

---

Mobile Device Management (MDM)

Mobile Device Management (MDM)

Definition: Software used to manage, monitor, and secure mobile devices.
Example: An organization remotely enforcing security settings on employee phones.

Device Configurations

Definition: Settings applied to mobile devices through management tools.
Example: Configuring corporate email automatically on employee devices.

Corporate Devices

Definition: Devices owned and managed by an organization.
Example: A company-issued smartphone for employees.

Bring Your Own Device (BYOD)

Definition: A policy allowing employees to use personal devices for work.
Example: An employee accessing company email on a personal phone.

Policy Enforcement

Definition: Applying organizational rules to managed devices.
Example: Requiring a passcode on all company phones.

Corporate Applications

Definition: Business applications installed and managed by an organization.
Example: Deploying Microsoft Teams through MDM.

---

Mobile Device Synchronization

Mobile Device Synchronization

Definition: The process of keeping data consistent across multiple devices and services.
Example: Synchronizing contacts between a smartphone and cloud account.

Recognizing Data Caps

Definition: Monitoring cellular data usage to avoid exceeding plan limits.
Example: Receiving a warning after using 90% of a monthly data allowance.

Calendar Synchronization

Definition: Keeping calendar events updated across devices.
Example: A meeting added on a laptop appearing on a smartphone.

Contacts Synchronization

Definition: Keeping contact information consistent across devices.
Example: A new contact added on a phone appearing in an email account.

Business Applications

Definition: Applications used to support business operations and productivity.
Example: Accessing company resources through Microsoft 365.

Mail

Definition: An application used to send, receive, and manage email messages.
Example: Synchronizing Outlook email across multiple devices.

Cloud Storage

Definition: Online storage services that allow files to be accessed from multiple devices.
Example: Accessing files stored in Google Drive from a smartphone and laptop.
