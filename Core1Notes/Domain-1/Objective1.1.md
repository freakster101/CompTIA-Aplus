Objective 1.1 - Mobile Device Hardware and Replacement Techniques

Battery

Definition: A rechargeable power source that provides energy to a mobile device.
Example: Replacing a swollen battery in a smartphone.

Keyboard/Keys

Definition: Input components used to enter text and commands on a mobile device.
Example: Replacing a non-functioning key on a laptop keyboard.

Random-Access Memory (RAM)

Definition: Temporary memory used by a device to store active data and applications.
Example: Upgrading a laptop from 8 GB RAM to 16 GB RAM.

Hard Disk Drive (HDD) / Solid-State Drive (SSD)

Definition: Storage devices used to permanently store operating systems, applications, and user data.
Example: Replacing an HDD with an SSD to improve performance.

Wireless Cards

Definition: Hardware components that provide wireless network connectivity.
Example: Replacing a faulty Wi-Fi card in a laptop.

---

Physical Privacy and Security Components

Biometrics

Definition: Security features that use unique physical characteristics for authentication.
Example: Unlocking a smartphone using a fingerprint scanner.

Near-Field Scanner Features

Definition: Components that use short-range wireless communication for authentication or payments.
Example: Using NFC to make a contactless payment with a smartphone.

---

Wi-Fi Antenna Connector/Placement

Definition: Antennas and connectors that transmit and receive wireless signals.
Example: Reconnecting a loose Wi-Fi antenna cable inside a laptop.

Camera/Webcam

Definition: An imaging device used to capture photos and video.
Example: Replacing a malfunctioning laptop webcam.

Microphone

Definition: An input device used to capture audio.
Example: Replacing a faulty microphone in a smartphone.
