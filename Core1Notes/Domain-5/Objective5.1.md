Objective 5.1 - Troubleshoot Motherboards, RAM, CPUs, and Power

Common Symptoms

POST Beeps

Definition: Audible beep codes generated during startup to indicate hardware problems.
Example: A continuous beep indicating a memory (RAM) failure.

Proprietary Crash Screens

Definition: Manufacturer-specific error screens displayed when a system encounters a critical problem.
Example: A Dell diagnostic error screen appearing during startup.

Blank Screen

Definition: A display that remains black despite the computer being powered on.
Example: A loose RAM module causing no video output.

No Power

Definition: A condition where a computer fails to turn on.
Example: A faulty power supply preventing system startup.

Sluggish Performance

Definition: Noticeably slow system operation or responsiveness.
Example: Insufficient RAM causing applications to run slowly.

Overheating

Definition: Excessive component temperatures that affect system stability.
Example: A CPU overheating due to a failed cooling fan.
	
Burning Smell

Definition: An odor indicating overheating or electrical component failure.
Example: A damaged power supply emitting a burning smell.

Random Shutdown

Definition: Unexpected system power-offs without user action.
Example: A CPU shutting down because of excessive heat.

Application Crashes

Definition: Programs closing unexpectedly due to hardware or software issues.
Example: Faulty RAM causing applications to crash repeatedly.

Unusual Noise

Definition: Abnormal sounds coming from computer components.
Example: A failing CPU fan producing grinding noises.

Capacitor Swelling

Definition: Physical bulging of capacitors caused by electrical failure.
Example: Swollen capacitors visible on a motherboard.

Inaccurate System Date/Time

Definition: Incorrect date or time settings after system startup.
Example: A depleted CMOS battery causing the clock to reset.
