Objective 5.5 - Troubleshoot Network Issues

Common Symptoms

Intermittent Wireless Connectivity

Definition: A wireless connection that frequently disconnects or drops unexpectedly.
Example: A laptop repeatedly disconnecting from Wi-Fi throughout the day.

Slow Network Speeds

Definition: Data transfers occurring at lower-than-expected speeds.
Example: Large files taking an unusually long time to download.

Limited Connectivity

Definition: A device connected to the network but unable to access all network resources or the internet.
Example: A computer connected to Wi-Fi but unable to browse websites.

Jitter

Definition: Variations in packet arrival times across a network connection.
Example: Choppy audio during an internet phone call.

Poor Voice over Internet Protocol (VoIP) Quality

Definition: Degraded voice communication caused by network issues.
Example: Echoes, delays, or robotic voices during a VoIP call.

Port Flapping

Definition: A network port repeatedly changing between active and inactive states.
Example: A switch port continuously disconnecting and reconnecting.

High Latency

Definition: Excessive delay in network communication between devices.
Example: Noticeable lag during an online video conference.

External Interference

Definition: Disruptions caused by external devices or environmental factors affecting network signals.
Example: A microwave causing Wi-Fi signal degradation.

Authentication Failures

Definition: Unsuccessful attempts to verify user or device credentials.
Example: A user entering an incorrect Wi-Fi password.

Intermittent Internet Connectivity

Definition: Internet access that works inconsistently and disconnects periodically.
Example: A router losing internet access several times each day.
