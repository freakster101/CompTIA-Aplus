Objective 5.2 - Troubleshoot Drive and RAID Issues

Common Symptoms

LED Status Indicators

Definition: Drive status lights that indicate activity, health, or failure conditions.
Example: A solid amber LED indicating a failed drive in a RAID array.

Grinding Noises

Definition: Unusual mechanical sounds coming from a hard disk drive.
Example: A failing HDD producing grinding sounds during operation.

Clicking Sounds

Definition: Repetitive clicking noises caused by a failing hard drive mechanism.
Example: An HDD making clicking noises and becoming inaccessible.

Bootable Device Not Found

Definition: An error indicating the system cannot locate a valid operating system.
Example: A failed SSD causing a "No Boot Device Found" message.

Data Loss/Corruption

Definition: Missing, damaged, or unreadable files stored on a drive.
Example: Documents becoming inaccessible after a drive failure.

RAID Failure

Definition: A condition where one or more drives cause a RAID array to become degraded or unavailable.
Example: A RAID 5 array entering a degraded state after a disk failure.

S.M.A.R.T. Failure

Definition: A warning generated when a drive detects internal reliability issues.
Example: A hard drive reporting imminent failure through S.M.A.R.T. monitoring.

Extended Read/Write Times

Definition: Slow access speeds when reading or writing data.
Example: Files taking an unusually long time to open from a failing drive.

Low Performance IOPS

Definition: Reduced input/output operations per second causing poor storage performance.
Example: A database server responding slowly due to storage bottlenecks.

Missing Drives in OS

Definition: Storage devices not appearing within the operating system.
Example: A disconnected SATA cable causing a drive to disappear from Windows.

Array Missing

Definition: A RAID array that is no longer detected by the RAID controller.
Example: A server failing to recognize a configured RAID 10 array.

Audible Alarms

Definition: Warning sounds generated by RAID controllers or storage systems.
Example: A RAID controller beeping continuously after a drive failure.
