Objective 3.5 - Motherboards, CPUs, and Add-on Cards

Motherboard Form Factors

Advanced Technology Extended (ATX)

Definition: A standard motherboard form factor commonly used in desktop computers.
Example: Building a gaming PC with an ATX motherboard.

microATX

Definition: A smaller motherboard form factor with fewer expansion slots than ATX.
Example: Installing a microATX motherboard in a compact desktop case.

Information Technology eXtended (ITX)

Definition: A compact motherboard form factor designed for small-form-factor systems.
Example: Building a mini PC using a Mini-ITX motherboard.

---

Motherboard Connector Types

Peripheral Component Interconnect (PCI)

Definition: A legacy expansion slot used for add-on cards.
Example: Installing an older sound card in a PCI slot.

PCI Express (PCIe)

Definition: A high-speed expansion interface used by modern add-on cards.
Example: Installing a graphics card in a PCIe x16 slot.

Power Connectors

Definition: Connectors that provide power from the power supply to the motherboard.
Example: Connecting a 24-pin ATX power connector.

SATA

Definition: A connector used to attach storage devices to a motherboard.
Example: Connecting a SATA SSD to the motherboard.

eSATA

Definition: An external version of SATA used for external storage devices.
Example: Connecting an external hard drive through an eSATA port.

Headers

Definition: Pins on a motherboard used to connect internal components and ports.
Example: Connecting front-panel USB ports to a USB header.

M.2

Definition: A motherboard connector used for compact storage and expansion devices.
Example: Installing an M.2 NVMe SSD.

---

Motherboard Compatibility

CPU Socket Types

Definition: Physical interfaces that connect CPUs to motherboards.
Example: Matching a CPU to a compatible motherboard socket.

AMD Socket

Definition: A CPU socket designed for AMD processors.
Example: Installing a Ryzen processor into an AM5 socket.

Intel Socket

Definition: A CPU socket designed for Intel processors.
Example: Installing a Core i7 processor into an LGA1700 socket.

Multisocket

Definition: A motherboard that supports more than one CPU.
Example: A server motherboard with two processor sockets.

---

BIOS/UEFI Settings

BIOS/UEFI

Definition: Firmware that initializes hardware and starts the operating system.
Example: Accessing UEFI settings during system startup.

Boot Options

Definition: Settings that determine the order of boot devices.
Example: Configuring a USB drive as the first boot device.

USB Permissions

Definition: Settings that control USB device access and functionality.
Example: Disabling USB ports for security purposes.

TPM Security Features

Definition: Hardware-based security settings provided by a Trusted Platform Module.
Example: Enabling TPM for BitLocker encryption.

Fan Considerations

Definition: Settings that control system fan operation and cooling performance.
Example: Adjusting fan speed profiles in UEFI.

Secure Boot

Definition: A security feature that allows only trusted software to load during startup.
Example: Preventing unauthorized operating systems from booting.

Boot Password

Definition: A password required before the operating system starts.
Example: Entering a password before Windows loads.

BIOS Password

Definition: A password that restricts access to BIOS/UEFI settings.
Example: Preventing users from changing boot settings.

Temperature Monitoring

Definition: A feature that tracks hardware temperatures.
Example: Viewing CPU temperatures in UEFI.

---

Virtualization Support

Virtualization Support

Definition: CPU and motherboard features that enable virtual machines.
Example: Enabling Intel VT-x or AMD-V in UEFI.

---

Encryption

Trusted Platform Module (TPM)

Definition: A hardware chip that stores cryptographic keys and security information.
Example: Using TPM to secure BitLocker encryption keys.

Hardware Security Module (HSM)

Definition: A dedicated device used to generate and protect cryptographic keys.
Example: An organization storing encryption keys in an HSM.

---

CPU Architecture

x86/x64

Definition: Processor architectures commonly used in desktop and server computers.
Example: Running Windows 11 on a 64-bit processor.

ARM

Definition: A processor architecture designed for power efficiency and mobile computing.
Example: A smartphone using an ARM-based processor.

Core Configurations

Definition: The number of processing cores contained within a CPU.
Example: An 8-core processor handling multiple tasks simultaneously.

---

Expansion Cards

Sound Card

Definition: An expansion card that provides audio processing capabilities.
Example: Installing a sound card for improved audio quality.

Video Card

Definition: An expansion card that processes graphics and video output.
Example: Installing a dedicated graphics card for gaming.

Capture Card

Definition: An expansion card used to record or stream video from external sources.
Example: Capturing gameplay from a gaming console.

Network Interface Card (NIC)

Definition: An expansion card that provides network connectivity.
Example: Installing a NIC to add Ethernet connectivity.

---

Cooling

Fans

Definition: Devices that move air to remove heat from system components.
Example: Installing a case fan to improve airflow.

Heat Sink

Definition: A metal component that dissipates heat from electronic devices.
Example: Attaching a heat sink to a CPU.

Thermal Paste/Pads

Definition: Materials that improve heat transfer between components and heat sinks.
Example: Applying thermal paste between a CPU and its cooler.

Liquid Cooling

Definition: A cooling system that uses liquid to transfer heat away from components.
Example: Using an all-in-one liquid cooler for a high-performance CPU.
