Objective 3.4 - Storage Devices

Hard Drives

Hard Disk Drive (HDD)

Definition: A storage device that uses spinning magnetic platters to store data.
Example: A desktop computer using a 1TB HDD for file storage.

Spindle Speeds

Definition: The rotational speed of an HDD measured in revolutions per minute (RPM).
Example: A 7200 RPM hard drive providing faster performance than a 5400 RPM drive.

Hard Drive Form Factors

2.5-inch

Definition: A smaller drive form factor commonly used in laptops.
Example: A 2.5-inch SSD installed in a notebook computer.

3.5-inch

Definition: A larger drive form factor commonly used in desktop computers.
Example: A 3.5-inch HDD installed in a desktop PC.

---

Solid-State Drives (SSDs)

Solid-State Drive (SSD)

Definition: A storage device that uses flash memory instead of spinning disks.
Example: A laptop using a 512GB SSD for faster boot times.

Communication Interfaces

Non-Volatile Memory Express (NVMe)

Definition: A high-speed storage protocol designed specifically for SSDs.
Example: An NVMe SSD installed in an M.2 slot.

SATA

Definition: A storage interface used to connect HDDs and SSDs to a computer.
Example: A SATA SSD connected to a motherboard.

Peripheral Component Interconnect Express (PCIe)

Definition: A high-speed interface used by modern expansion cards and storage devices.
Example: An NVMe SSD communicating through PCIe lanes.

Serial Attached SCSI (SAS)

Definition: A high-performance storage interface commonly used in enterprise servers.
Example: A SAS drive installed in a data center server.

SSD Form Factors

M.2

Definition: A compact SSD form factor that connects directly to a motherboard.
Example: Installing an M.2 NVMe SSD in a laptop.

mSATA

Definition: A smaller SSD form factor based on the SATA interface.
Example: Upgrading storage in an older ultrabook using mSATA.

---

Drive Configurations

RAID 0

Definition: A RAID configuration that stripes data across drives for increased performance without redundancy.
Example: Two drives combined to improve read and write speeds.

RAID 1

Definition: A RAID configuration that mirrors data across two drives for redundancy.
Example: One drive automatically copying all data to a second drive.

RAID 5

Definition: A RAID configuration that uses striping with parity for fault tolerance.
Example: Three drives storing data with parity protection.

RAID 6

Definition: A RAID configuration that uses double parity for increased fault tolerance.
Example: A storage array surviving the failure of two drives.

RAID 10

Definition: A RAID configuration that combines mirroring and striping for performance and redundancy.
Example: Four drives configured as mirrored pairs with striped data.

---

Removable Storage

Flash Drive

Definition: A portable storage device that uses flash memory and connects through USB.
Example: Transferring files using a 64GB USB flash drive.

Memory Card

Definition: A removable flash storage device used in cameras, phones, and other portable devices.
Example: Storing photos on a microSD card.

---

Optical Drives

Optical Drive

Definition: A device that reads and/or writes data using optical discs.
Example: Installing software from a DVD drive.
