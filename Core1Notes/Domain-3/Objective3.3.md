Objective 3.3 - RAM Characteristics

Form Factors

Small Outline Dual In-line Memory Module (SODIMM)

Definition: A compact RAM module commonly used in laptops and small-form-factor devices.
Example: Installing a SODIMM module in a laptop.

Dual In-line Memory Module (DIMM)

Definition: A standard-sized RAM module commonly used in desktop computers.
Example: Upgrading a desktop PC with a DIMM memory module.

Double Data Rate (DDR) Iterations

DDR Iterations

Definition: Successive generations of RAM technology that provide increased speed and efficiency.
Example: Replacing DDR4 memory with DDR5 memory in a newer system.

Error-Correcting Code (ECC) vs. Non-ECC RAM

Error-Correcting Code (ECC) RAM

Definition: RAM that can detect and correct certain memory errors automatically.
Example: A server using ECC RAM to improve data reliability.

Non-ECC RAM

Definition: RAM that does not provide automatic error detection and correction.
Example: A home desktop computer using non-ECC memory.

Channel Configurations

Single Channel

Definition: A memory configuration that uses one communication channel between RAM and the CPU.
Example: A system with one RAM module installed.

Dual Channel

Definition: A memory configuration that uses two channels to increase memory performance.
Example: Two identical RAM modules installed in matching slots.

Triple Channel

Definition: A memory configuration that uses three channels for increased memory bandwidth.
Example: Three matching RAM modules installed in a supported motherboard.

Quad Channel

Definition: A memory configuration that uses four channels to maximize memory throughput.
Example: Four identical RAM modules installed in a workstation motherboard.
