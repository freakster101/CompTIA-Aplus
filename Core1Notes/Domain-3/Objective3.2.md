Objective 3.2 - Cable Types, Connectors, Features, and Purposes

Network Cables

Copper Cable

Definition: A networking cable that transmits data using electrical signals.
Example: Connecting a computer to a switch using Ethernet cable.

Categories (Cat5e, Cat6, Cat6a, etc.)

Definition: Standards that define the performance and speed of twisted-pair cables.
Example: Using Cat6 cable for a Gigabit network.

T568A/T568B Standards

Definition: Wiring standards that specify the order of wires in Ethernet connectors.
Example: Terminating both ends of a cable using T568B.

Coaxial Cable

Definition: A cable with a central conductor surrounded by shielding for signal transmission.
Example: Connecting a cable modem to an ISP connection.

Shielded Twisted Pair (STP)

Definition: Twisted-pair cable with shielding to reduce electromagnetic interference (EMI).
Example: Using STP cable in an industrial environment.

Direct Burial Cable

Definition: A cable designed to be installed underground without conduit.
Example: Running network cable between two buildings underground.

Unshielded Twisted Pair (UTP)

Definition: Twisted-pair cable without additional shielding.
Example: Using Cat6 UTP cable in an office network.

Plenum-Rated Cable

Definition: A cable designed for installation in air-handling spaces with fire-resistant materials.
Example: Running network cable through a building's ceiling plenum.

Optical Cable

Definition: A cable that transmits data using light instead of electricity.
Example: Connecting network switches with fiber-optic cable.

Single-Mode Fiber

Definition: Fiber-optic cable designed for long-distance communication using a single light path.
Example: Connecting two campuses several kilometers apart.

Multimode Fiber

Definition: Fiber-optic cable designed for shorter distances using multiple light paths.
Example: Connecting switches within a data center.

---

Peripheral Cables

USB 2.0

Definition: A USB standard supporting data transfer speeds up to 480 Mbps.
Example: Connecting a keyboard to a computer.

USB 3.0

Definition: A USB standard supporting significantly faster data transfer speeds than USB 2.0.
Example: Connecting an external SSD.

Serial Cable

Definition: A cable used for serial communication between devices.
Example: Configuring a network switch through a console port.

Thunderbolt

Definition: A high-speed interface supporting data, video, and power transmission.
Example: Connecting a docking station to a laptop.

---

Video Cables

HDMI

Definition: A cable that carries digital audio and video signals.
Example: Connecting a monitor to a computer.

DisplayPort

Definition: A digital display interface commonly used with computers and monitors.
Example: Connecting a gaming monitor to a graphics card.

DVI

Definition: A video interface used to transmit digital video signals.
Example: Connecting an older monitor to a desktop PC.

VGA

Definition: An analog video interface used on legacy display devices.
Example: Connecting an older projector to a laptop.

USB-C

Definition: A reversible connector capable of carrying data, video, and power.
Example: Connecting a laptop to a monitor using USB-C.

---

Hard Drive Cables

SATA

Definition: A cable standard used to connect storage devices to a motherboard.
Example: Connecting an SSD to a desktop computer.

eSATA

Definition: An external version of SATA used for external storage devices.
Example: Connecting an external hard drive through an eSATA port.

---

Adapters

Adapter

Definition: A device used to connect incompatible cable or connector types.
Example: Using a USB-C to HDMI adapter.

---

Connector Types

RJ11

Definition: A connector commonly used for telephone lines.
Example: Connecting a DSL modem to a phone jack.

RJ45

Definition: A connector commonly used for Ethernet networking.
Example: Connecting a computer to a switch.

F-Type

Definition: A connector commonly used with coaxial cables.
Example: Connecting a cable modem to a coaxial outlet.

Straight Tip (ST)

Definition: A fiber-optic connector that uses a twist-lock mechanism.
Example: Connecting fiber cables in legacy networks.

Subscriber Connector (SC)

Definition: A fiber-optic connector that uses a push-pull design.
Example: Connecting fiber patch cables to network equipment.

Lucent Connector (LC)

Definition: A small-form-factor fiber-optic connector.
Example: Connecting fiber transceivers in a switch.

Punchdown Block

Definition: A termination block used for network and telephone cabling.
Example: Terminating Ethernet cables on a patch panel.

MicroUSB

Definition: A small USB connector commonly used on older mobile devices.
Example: Charging an older Android smartphone.

MiniUSB

Definition: A compact USB connector used on older portable devices.
Example: Connecting a digital camera to a computer.

USB-C

Definition: A modern reversible USB connector for power, video, and data.
Example: Charging a laptop with a USB-C cable.

Molex

Definition: A power connector commonly used in desktop computers.
Example: Providing power to an older hard drive.

Lightning

Definition: Apple's proprietary connector used on many Apple mobile devices.
Example: Charging an iPhone.

DB9

Definition: A 9-pin serial connector used for legacy communication devices.
Example: Connecting to a router's console port.
