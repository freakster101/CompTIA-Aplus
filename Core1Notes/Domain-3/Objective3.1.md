Objective 3.1 - Display Components and Attributes

Display Types

Liquid Crystal Display (LCD)

Definition: A display technology that uses liquid crystals and a backlight to produce images.
Example: A standard laptop screen using LCD technology.

In-Plane Switching (IPS)

Definition: An LCD panel type known for excellent color accuracy and wide viewing angles.
Example: A graphic designer using an IPS monitor for photo editing.

Twisted Nematic (TN)

Definition: An LCD panel type known for fast response times and lower cost.
Example: A gamer using a TN monitor for competitive gaming.

Vertical Alignment (VA)

Definition: An LCD panel type known for high contrast ratios and deep blacks.
Example: Watching movies on a VA television display.

Organic Light-Emitting Diode (OLED)

Definition: A display technology where each pixel produces its own light.
Example: A smartphone with an OLED screen displaying true blacks.

Mini-LED

Definition: A display technology that uses thousands of tiny LEDs for improved brightness and contrast.
Example: A high-end laptop using a Mini-LED display.

---

Display Components

Touch Screen

Definition: A display that accepts user input through touch.
Example: Navigating a smartphone using finger gestures.

Digitizer

Definition: A layer that converts touch input into digital signals.
Example: A tablet detecting stylus input through a digitizer.

Inverter

Definition: A component that supplies power to the backlight in older LCD displays.
Example: Replacing a faulty inverter in a laptop with a dim screen.

---

Display Attributes

Pixel Density

Definition: The number of pixels packed into a display area, usually measured in PPI.
Example: A smartphone with 460 pixels per inch (PPI).

Refresh Rate

Definition: The number of times a display updates its image each second.
Example: A gaming monitor operating at 144Hz.

Screen Resolution

Definition: The number of pixels displayed on a screen.
Example: A monitor with a resolution of 1920×1080.

Color Gamut

Definition: The range of colors a display can reproduce.
Example: A monitor covering 100% of the sRGB color space.
